Trình soạn thảo HTML

Bạn chỉ cần một công cụ cơ bản để lập trình HTML.

Phần mềm học HTML (NotePad)

Để bắt đầu học HTML hoặc bạn là người mới, hãy bắt đầu bằng NotePad trên Windows.


Để mở NotePad bạn cần ấn thanh tìm kiếm trên Taskbar rồi tìm NotePad.


Viết mã HTML

Hãy thử viết hoặc sao chép đoạn mã HTML sau.

Example
<!DOCTYPE html>
<html>
<head>
  <title>Tiêu đề trang </title>
</head>
<body>
  <h1>Heading </h1>
  <p>Đoạn paragraph </p>
</body>
</html>

Lưu trang HTML

Lưu tệp, bạn ấn Ctrl + S hoặc chọn File -> Save As trong menu NotePad.


Đặt tên tệp là "index.html" và thiết lập mã hóa thành UTF-8 (đây là mã hóa được khuyến nghị cho các tệp HTML).


Thử xem trang HTML trong trình duyệt

Thử mở tệp HTML bạn đã lưu.


Kết quả sẽ trông như này.



Bạn có thể thấy được title và nội dung trong trình duyệt của bạn.
